## Day 3 | 6/19（五·端午）— 🎢 环球影城全天

> **环球影城调到端午第一天，建议买全项优速通。**

### 🚇 交通
🚇2号线 → 🚇7号线「环球度假区」站，约50min

### ⏰ 时间线

| 时间 | 安排 | 备注 |
|:----|:------|:------|
| **08:00** | 到环球影城门口 | 地铁站出来走5min |
| **08:30** | **🎢 入园** | 提前APP预约，刷身份证 |
| **08:30-10:00** | **🎯 第一波：冲哈利波特·禁忌之旅** 🥇 | 进园直冲不要停！ |
| 10:00-11:00 | **霸天虎过山车** 🥈 | 变形金刚区，上午排队短 |
| 11:00-12:00 | **侏罗纪世界大冒险** 🥉 | |
| 12:00-12:30 | 小黄人乐园 | 轻松项目+拍照 |
| **12:30-14:00** | **😴 午休 + 午餐** | 三把扫帚/平先生面馆 |
| **14:00-15:00** | **🌊 水世界表演 ⭐必看** | 特技+爆破，坐前排会湿身 |
| 15:00-17:00 | 补刷+互动 | 变形金刚传奇现场、功夫熊猫区 |
| 17:00-18:00 | 哈利波特二刷 | 如果还想玩禁忌之旅 |
| 18:30-19:30 | **🎉 花车巡游** | 环球大巡游 |
| **20:00-20:30** | **🌙 哈利波特城堡灯光秀 🔥必看** | 不看完等于白来 |
| 20:30后 | 出园回酒店 | |

### 🎯 必玩排名
1. **哈利波特·禁忌之旅** 🥇 — 进园第一件事
2. **霸天虎过山车** 🥈 — 最刺激
3. **侏罗纪世界大冒险** 🥉 — 沉浸式恐龙
4. **水世界表演** — 特技爆破必看
5. **哈利波特城堡灯光秀** — 晚上才有

### 💡 省时间技巧
- **全项优速通**（端午人多建议买）
- 或者至少单项优速通给哈利波特
- 下载「北京环球度假区」APP看实时排队
- 带空水瓶，园区有直饮水
