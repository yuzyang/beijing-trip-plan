## Day 2 | 6/18（四·工作日）— ⭐升旗 · 🏛️故宫 · ⛰️景山 · 🏯北海 · 🛍️王府井

### 🚇 地铁
前门/崇文门 → 🚇1号线天安门东/西

### ⭐ 升旗（~04:45）
**📍 最佳位置：** 金水桥前200-300m，或长安街中段
**⏰ 时间策略：** 03:30到安检 → 04:10-04:40看升旗 → 05:30回酒店补觉

### 😴 补觉 + 早餐（05:30-09:00）
回前门酒店睡觉，睡到自然醒再出门

### 🏛️ 故宫博物院（09:00-12:00）

- 07:30前到午门排队 → 08:30开门直接冲中轴线
- 只逛中轴线+珍宝馆，3h精华不贪多

**📸 拍照点：**
- 午门正面
- 太和殿台阶（人最多但最经典）
- 御花园角落（人少）

### ⛰️ 景山公园（13:30-14:10）
门票2元，午门对面走过去5min
**📸 最佳机位：** 万春亭拍故宫全景（最经典北京照片）

### 🏯 北海公园（14:30-15:30）
门票10元，景山走过去5min
白塔+湖景，可划船（夏天很舒服）

### 🛍️ 王府井（16:00-17:30 · 可选）
北海走过去15min，累了就跳过直接回酒店

## Day 6 | 6/22（一·工作日）— 🎢 环球影城全天

> 周一工作日人最少。提前买好门票+优速通。

### 🚇 交通
🚇7号线/八通线直达「环球度假区」站
从你住的前门出发：🚇2号线 → 🚇7号线，约50min

### ⏰ 最优时间线

| 时间 | 安排 | 备注 |
|:----|:-----|:------|
| 08:00 | 到环球影城门口 | 地铁站出来走5min |
| 08:30 | **入园** | 提前APP预约，刷身份证/二维码 |
| 08:30-10:00 | **🎯 第一波：冲哈利波特** | 🚶 进园直冲不要停！优先玩禁忌之旅 |
| 10:00-11:00 | **霸天虎过山车** | 变形金刚区，上午排队短 |
| 11:00-12:00 | **侏罗纪世界大冒险** | |
| 12:00-12:30 | **小黄人乐园** | 轻松项目，顺便拍照 |
| 12:30-14:00 | **😴 午休 + 午餐** | 建议三把扫帚（哈利波特区）或平先生面馆 |
| 14:00-15:00 | **水世界表演 🌊** | ⭐ 必看！特技表演，坐前排会湿身 |
| 15:00-17:00 | **补刷 + 互动** | 变形金刚传奇现场、功夫熊猫区 |
| 17:00-18:00 | **哈利波特二刷** | 如果禁忌之旅还想玩一次 |
| 18:30-19:30 | **花车巡游** 🎉 | 环球大巡游，气氛很好 |
| 20:00-20:30 | **🌙 哈利波特城堡灯光秀** 🔥 | ⭐ 必看！不看完等于白来 |
| 20:30后 | 出园回市区/去机场 | |

### 🎯 必玩项目排名
1. **哈利波特·禁忌之旅** 🥇 — 进园第一件事
2. **霸天虎过山车** 🥈 — 最刺激的过山车
3. **侏罗纪世界大冒险** 🥉 — 沉浸式恐龙
4. **水世界表演** — 特技+爆破，必看
5. **哈利波特城堡灯光秀** — 晚上才有的氛围

### 💡 省时间技巧
- 买**全项优速通** → 热门项目几乎不用排队，约400-600元
- 或者买**单项优速通给哈利波特**（排队最长的）
- 单人通道：有些项目有单人通道，比普通排队快很多
- 提前下载「北京环球度假区」APP，看实时排队时间
- 带个空水瓶，园区有直饮水

### 📸 拍照点
- 哈利波特区霍格沃茨城堡（正门+湖边倒影）
- 小黄人乐园（色彩丰富，很出片）
- 好莱坞大道傍晚灯光

**提前订：** 官方小程序/APP买门票 + 优速通（建议至少单项给哈利波特）
